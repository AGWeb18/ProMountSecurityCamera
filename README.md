# ProMount Security Cameras Web

A professional landing page for **ProMount Security Cameras**, a security camera installation service serving the Durham Region and Kawartha Lakes in Ontario, Canada.

## Overview

This website is designed to convert local traffic into service leads. It highlights the business's core value propositions:
- **No Monthly Fees:** Emphasis on local recording and ownership of data.
- **Inclusive Service:** Tailored for both residential properties and commercial businesses.
- **Wired & Wire-Free:** Flexible installation options including PoE and Solar-powered systems.
- **Local Expertise:** Specifically targeted at Ontario communities like Whitby, Oshawa, Peterborough, and Lindsay.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Analytics:** Vercel Analytics
- **SEO:** Optimized with JSON-LD Structured Data (Schema.org) for LocalBusiness.

## Key Features

- **SEO Optimized:** Machine-readable metadata to improve ranking in Google Maps and local search results.
- **Tiered Pricing:** Transparent, all-inclusive packages (Basic, Standard, Premium).
- **Responsive Design:** Fully mobile-optimized for calls-to-action on the go.
- **Grandpa-Friendly Narrative:** A relatable "Our Story" section focused on peace of mind and family care.
